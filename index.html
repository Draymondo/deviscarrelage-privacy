Politique de confidentialité — DevisCarrelage

Dernière mise à jour : mai 2026

1. Collecte de données

DevisCarrelage ne collecte aucune donnée personnelle. L'application fonctionne entièrement hors ligne. Toutes les informations saisies (nom du client, coordonnées, devis) sont stockées uniquement sur votre appareil et ne sont jamais transmises à un serveur externe.

2. Stockage des données

Les données sont stockées localement sur votre appareil via AsyncStorage. Elles ne quittent jamais votre téléphone sauf si vous choisissez de partager un PDF manuellement.

3. Partage des données

DevisCarrelage ne partage aucune donnée avec des tiers. Le partage de PDF est une action volontaire de l'utilisateur via les applications installées sur son appareil.

4. Permissions

L'application peut demander l'accès au stockage de l'appareil uniquement pour sauvegarder et partager les PDF générés.

5. Contact

Pour toute question concernant cette politique, contactez-nous à l'adresse indiquée sur le Play Store.
